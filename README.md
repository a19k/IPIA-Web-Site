# IPI Akademija Website

This repository is a project of creating a website to test the knowledge and skills in following fields: HTML, CSS and JavaScript.
It is the first assignment in the subject of Web programming and it represents 20% of the grade.

## Installation
You can test the page using the GitHub preview : 

https://htmlpreview.github.io/?https://github.com/a19k/IPIA-Web-Site/blob/master/index.html

or you can clone the repository to your device using the terminal:

_git clone https://github.com/a19k/IPIA-Web-Site.git_

### Whiteboard
Whiteboard is a page which uses the Canvas API to create a simple drawing board. The behavior of the canvas was defined in JavaScript. User can change brush size and color, or can choose to use the eraser, print a .png file of the drawing or save the drawing for persistent usage. The save state can be deleted with a broom button which also cleans up the board.
<img width="4096" height="2016" alt="image" src="https://github.com/user-attachments/assets/b4002bd5-d880-4e10-9d3d-c2d3babb7206" />

### Visionboard
Visionboard is a page which serves as a personal board for editable sticky notes, quotes and images. Save button allows to save the board state for persistent usage. Broom button deletes the save along with everything on the board.
